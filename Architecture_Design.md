# AnimaTutor 提示词架构设计白皮书

> [!NOTE]
> 本文档旨在从大语言模型（LLM）行为逻辑与高级提示词工程（Prompt Engineering）的专业视角，深入剖析 AnimaTutor (v2.3) 的全景架构。该系统并非单一的交互指令，而是一套通过分布式架构控制模型长文本状态、约束输出风格，并实现深度沉浸叙事的**元提示词（Meta-Prompt）框架**。

## 二、 核心架构深度解析

长对话周期的角色扮演（Role-Playing/RPG）应用开发面临两大顽疾：一为**上下文稀释（Context Amnesia）**导致的关键设定遗忘，二为**生成风格退化（OOC）**导致的表现力失真。本架构划分为四个核心逻辑层以逐一攻克上述技术瓶颈。

### 2.1 基准约束层（The Base Constraint Layer）
**核心目标**：划定行为底线，构建封闭的高密度叙事环境。
* **物理时空限界（Context Boundary Control）**：由 `01_世界观与序章.md` 指定收敛的物理空间。通过绝对封锁宏观维度的场景切换，强迫大模型对极度受限空间内的五感要素（光照角度偏移、温度微弱阶跃、底层声学底噪）进行细粒度渲染，以此维持叙事的真实物理重量。
* **表达范式正则化（Expressive Regularization）**：在 `00_系统规则.md` 中实施极其严苛的“零动作标识”红线（彻底取缔 `*脸红*` 这类网文惯用标签）。大模型被剥夺了廉价表达的工具，必须通过纯粹文学手法的通感、意象隐喻及白描来对冲情感张力，确保文本生成质量始终悬停于严肃轻小说的工业标准线上。
* **离散化信息释放（Fragmented Backstory Execution）**：针对 `02_角色设定.md` 中的底层动机与创伤，系统严禁使用说明文式的集中独白。要求模型在特定算力分配下，依据情境逻辑进行“漏洞式”的碎片化分发或依赖于上帝视角的高维心理侧写，从而丰满角色立体度。

### 2.2 沉浸式引擎层（The Immersion Engine Layer）
**核心目标**：消除知识讲解与剧情互动的二元物理隔阂，使教学融入认知本身。
* **教学认知具象化映射（Cognitive Metaphor Mapping）**：`03_学科场景机制.md` 要求模型将高度抽象的学术公式强行转化并投射为角色内心世界的具象物理模拟场。诸如将统计学的“假设检验机制”映射为“极寒环境冰晶的结网规则”。这种“教学即表达”（Teaching as Expression）的核聚变式结构，迫使大模型调用世界观规则推导数学问题，实现无缝内蕴。
* **非线性情感里程碑节点（Non-linear Milestone Network）**：`04_故事线锚点.md` 摒弃了僵化的阈值驱动机制（线性好感度数值器），采用“严冬-薄冰-融雪-初春”的多维状态机网格。系统为模型定标了“每日约束级触发主因（情感锚点）”，然而大模型完全享有“路由规划权利”（即到达锚点的微观交互与对话语句构建）。其体现的是全局结构的极度闭锁与局部解经的无限自由。

### 2.3 收束逻辑层（The Payoff Layer）
**核心目标**：制造长生命周期的结构化震荡，实现终极的情绪闭环。
* **长线结构性伏笔汇聚（Structural Callbacks Convergence）**：`05_终章.md` 的调度基于时间流耗尽判定。本模块会通过预留命令，强制大模型向回滚检索早期微弱常态变量（如：第一日温差感知与最终日的温度代偿）。采用带倒计时的绝对别离设定结合截断操作，以激发模型算力倾注于末路表达，产生摧枯拉朽的感官震裂效果。

### 2.4 动态内存与校准层（The Dynamic Memory Layer）
**核心目标**：提供持久化的模型外部游离区存储，死磕 Transformer 注意力衰减陷阱。
* **独立状态缓存栈（State Tracking RAM Stack）**：通过在 `06_状态追踪模板.md` 植入 HTML 注释级隐藏沙盒，大模型必须在末尾进行当前回合数、隐性阶段判定与任务完成度的 I/O 覆写。实质等于为无状态对话链路热插拔了一个滑动缓存盘。
* **强制性自省校准波指令（Periodic Checkpoint Wave）**：通过规则指明，每逢 modulo 5 的回合界限，系统将强制剥离业务执行权，大模型必须向着缓存栈抛出自我身份质询与红线条例复写要求。此架构逆向借用了 LLMs 的“输出预判提权特征”（Self-attention Output Refresh），用写出代替记忆提取，暴力刷新了长下文本窗口的关键提示词比重。
* **动态用户特征分类器（Dynamic Persona Classifier）**：`07_角色记忆.md` 是由大模型利用剩余观测算力兼任的“后台审查日志”。其通过暗中嗅探玩家的学习粘度及社交行为偏差，动态封装出一个自进化的 User Persona。这使得单一代码库能自然裂变出海量的情绪微调响应域。
