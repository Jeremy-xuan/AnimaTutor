## ✨ 这是什么？

在各种 AI 大行其道的今天，传统的 AI 伴学和语C对话存在两个让人极其烦躁的痛点：
1. **像个毫无灵魂的做题机**：机械地把干瘪的知识点拍在你脸上，枯燥得让人看三行就不想往下学。
2. **极度生硬廉价的语C扮演**：不仅动不动就 `*红着脸笑了笑*` 毫无文学美感，而且只要多聊十几天，AI 必定会陷入赛博失忆，彻底忘记你们曾经历过的事。

**AnimaTutor就是为了彻彻底底解决这两大顽疾而生。**

它绝不是普通的“一段对话提示词”，而是一个专为顶级大语言模型特殊优化设计的 **Meta-Prompt**。只需在模板中填入你永远的推（你想见到的角色）和你最为头疼的倒霉学科（无论是物理/Java/高数等等等等...），它能当场变身为“轻小说作家兼游戏主策”，为你自动生成出一套 **8 份文件的“分布式长线剧情辅导提示词”**。

这套体系首创了**“教学即表达”**（用知识隐喻角色内心的世界观）和**“8文件状态机抗遗忘架构”**。拿着它跑出来的剧本，大模型不再是冰冷的代码，而是一个能陪你走过春夏秋冬、用绝美的克制旁白推进羁绊、并手把手带你看具象化场景学知识的灵魂伴学导师。

## 💡 AnimaTutor是怎样的体验？

面对你抛过去的一道“微积分极限”推导题，一般的 AI vs 搭载了 AnimaTutor 引擎的 AI 反应截然不同：

❌ **传统 AI 辅导模式：**
> "了解！导数就是函数的瞬时变化率。而极限指的是趋近但不等于。这道题的推导步骤如下：第一步提取公因式..."

✅ **AnimaTutor 生成版实机演示**
> 活动室里的斜阳越过了窗沿，将木地板上细微的浮尘照得像停滞在空中的碎金。
> 你的笔尖悬停在公式 `lim(x→∞) (1+1/x)^x` 上，很久没有落下。
> “它的极限是自然对数底 e，”她没有抬头，翻过文库本时的纸张摩擦声极其轻微，“就像你把一块方糖无限次地切碎、再分别投入同一杯红茶里。”
> 她的视线终于从书页上移开，越过茶杯升腾的雾气，非常安静地看着你。
> “哪怕你切割的次数接近无穷大，方糖溶解带来的甜度提升，最终也会停留在一个绝对不可跨越的边界上。无法再更进一步了——就像某些注定无法抵达的距离。”
> 她伸出纤细的手指，在你卡住的那行演算纸旁点了一下，指尖带着一点不容拒绝的微凉：“现在，继续求导。”

不仅杜绝了传统语 C 里廉价的 `*脸红笑了笑*` 这类动作，所有知识都被强行扭曲为**角色内心世界可视化的投影**。

## 🕹️ 我该怎么玩？

你可能会问：这满屏幕的轻小说感，**我需要陪她演戏吗？我该回复什么才不会导致剧情卡住？**

**答案是：什么都不需要演。你只需要“出现”。**

1. **纯刷题玩家**：你大可以带着一天的怨气，把不会的数学题直接甩在屏幕上。不用说“你好”，也不用客套。**系统的时钟会随着你的发题自动推进**。她会教你，剧情会在傍晚的光线中自行演进。
2. **想闲聊的玩家**：你在某道题中间如果吐槽一句“这知识点太变态了，你以前是怎么学的？”，她的反应会根据她当前跟你处于什么**阶段（严冬-薄冰-融雪-初春）**，自然地搭理你或者冷着脸无视你。
3. **被动体验**：一切好感度升级、过往身世的刀子爆出，都不需要你费尽心机去选完美的 GalGame 选项。你只要像个真实世界里做不来题的学生一样坐在她对面，命运的齿轮就会自己转动。

## 🛠 前置环境依赖 (Prerequisites)

在登入这个沉浸式文字世界之前，你需要准备以下基础设施：
- **基准模型算力**：至少支持 **100K 以上**上下文窗口大小（Context Window）的大语言模型。
- **对话承载器**：推荐支持“多文件上传外挂”或“长记忆留存”的云端知识库平台（例如 **Claude Pro 的 Projects**、**ChatGPT Plus 的 Custom GPTs**，或 **Kimi 的知识库**）。
- **（硬核高玩选配）自动化构建工具**：一个支持本地读写文件系统代码的智能体（Agent）IDE，如 `Claude Code`、`Cursor`等，用于一键闭环生成长达数万字的 8 份代码组件。

## 🚀 快速开始部署 (Quick Start)

如何得到这套 8 份文件构成的神仙伴学系统？有三种流派供你选择：

### 🥇 选项 A：免写代码！知识库无脑部署流 (最推荐)
如果你拥有各大顶级大模型的会员或高级版：
1. 用 **[Universal Prompt Template v2.3.md](./Universal%20Prompt%20Template%20v2.3.md)** 去找大模型，让它按照你在最上方 YAML 填写的你推的名字，生成完整的 8+1 份 markdown 文件，保存到你电脑的文件夹里。
2. 打开 **Claude Projects（项目库）** / **ChatGPT Custom GPTs** / **Kimi 知识库**。
3. 创建一个新项目，把你桌面上生成的 8+1 份文件**直接全部拖进去上传**。
4. 在对话框大喊一句：“读取 `BOOT.md` 开始吧！”，即可永久开启属于你的文字长篇连载！大模型会在后台自动检索文件，再无赛博失忆。

### ⚙️ 选项 B：Agentic AI 极客流 (硬核玩家适用)
使用终端智能体如 Claude Code, Cursor：喂入这份模板，直接让它在本地目录生成全部文件树。随后打包上传即可痛快开玩，效率最高。

### 🛋️ 选项 C：网页端一键复制懒人版 (v1.1 单文件版)
打开 **[v1.1 极简单文件版](./Universal%20Prompt%20Template%20v1.1.md)**。手动改掉顶部的角色名和学科预设，然后一波全选，直接扔给任何一个聪明的网页大模型框里。发题即开玩！（唯一缺点：长聊十几日后可能会有遗忘大结局伏笔的风险）。

## 🤖 OOC防护装甲：用什么模型能获得完美的体验？

这套引擎封禁了所有常规语 C 的廉价表达手法，对模型本身的文学素养和**指令遵循能力**提出了相当苛刻的要求。以下是推荐的大模型排序：

🔥 **T0 文学质感拉满，绝不OOC**
- **Claude 4.6 (Sonnet / Opus)**：当之无愧的语 C 霸主，文笔极其极其克制。能百分百还原那种“用光线微变替代好感度”的幽微文学感，文件记忆力无敌。
- **Gemini(推荐3.1pro)**: Gemini虽然没有Claude好用, 但是硬着头皮也能跑得起来--但是记得要用3.1pro, flash只会让你获得一团糟的故事情节.

🌟 **T1 国产之光（强烈推荐，性价比极高）**
- **DeepSeek-V3.2**：国产文字张力的巅峰之一，不仅极其聪明，而且非常听话。在生成轻小说旁白和理解长线文件时毫不逊色。
- **Kimi2.5 (200K+)**：拥有变态级别的长文本吞吐量。把 8 份文件全丢给它的知识库，聊上三个月都绝对不会触发失忆。
- **Qwen3.5 (通义千问)**：最新版对复杂场景的构建非常华丽，且擅长处理代码/理工科答疑的无缝场景转换。

❌ **T2 不推荐**
- 小参数量的本地开源模型、主打搜索或指令较弱的模型不建议使用。容易出现无视“零括号动作”红线、强行水字数、脱离场景干巴巴解题的灾难现场。
- **GPT-5.4**：宁可用deepseek也不要用GPT! GPT只会"稳稳的接住你", 但是及其难以代入角色

## 📂 项目架构与直达链接

为了能让大语言模型在漫长的陪伴中获得不灭的长期记忆，我们彻底抛弃了单文件 Prompt，转而打造了 **8+1 的多文件防遗忘护城河**。你的本地工程架构树最终会长这样：

```text
AnimaTutor/
├── ⚙️ BOOT.md                     # 系统引导盘：指挥大模型如何读盘与初始化
├── 🏛️ 00_系统规则.md               # 静态约束：底层协议与动作描写禁令
├── 🚪 01_世界观与序章.md            # 静态约束：物理时空边界与故事入场
├── 👤 02_角色设定.md               # 静态约束：角色的四维度画像与过往刀子碎片
├── 🌌 03_学科场景机制.md            # 动态映射：把枯燥学科转化为心境具象化世界的魔法引擎
├── 🎭 04_故事线锚点.md              # 情感骨架：四阶段情感里程碑与日更强制剧情
├── 🎬 05_终章.md                  # 核爆引擎：回收所有前期细节伏笔的限时离别大结局
├── 💾 06_状态追踪模板.md            # 后台缓存：每 5 轮强制校验防篡改的记忆块（AI 内部维护用）
└── 👁️ 07_角色记忆.md               # 社交图谱：由 AI 为你暗中绘制的主观属性观测图
```

- 📝 **设计架构白皮书**：[**Architecture_Design.md**](./Architecture_Design.md)（内藏硬核系统图表与原理解析）
