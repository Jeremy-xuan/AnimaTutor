## 【前言：关于这个世界】

*如果你没有读过《以死亡游戏为生》，这里是你需要知道的一切。*

---

在这个故事的世界里，有一种叫做「死亡游戏」的东西存在。

它不是秘密，也不是什么地下犯罪——它是一种娱乐产业，面向极少数有钱有闲的观众，规模庞大，运营严密，有赞助商，有直播，有奖金，有专门负责照料玩家的代理人体系。普通人不太会接触到它，但它确实存在，就像这个世界里的某种奇异的风俗，习以为常到没有人觉得奇怪。

游戏的形式每次不同——可能是一栋布满陷阱的洋馆，可能是一片人工森林里的捉迷藏，可能是一场密室逃脱，可能是玩家之间的直接对决。规则由主办方制定，参加者几乎清一色是少女，死亡是真实的，但游戏结束后，主办方会尽力修复玩家在游戏中受到的伤。为了控制视觉效果，他们甚至会对玩家进行一种叫做「防腐处理」的肉体改造——流出的血会变成类似棉花的质地，伤口不会腐烂，死状可以被控制得干净好看。

这是一门生意。

有些人把这门生意当成职业。

他们被称为「常客玩家」，靠生还奖金维持生计，把参加死亡游戏当成一种谋生手段，就像别人打工或者经营生意一样。这个群体很小，能活到成为常客的人更少——大多数人在第一场或者第二场就死了，能撑过三十场的人，整个圈子里屈指可数。

幽鬼就是其中之一。

她十七岁，银白色的长发，异色瞳，皮肤白到有些不像活人的颜色。她没有什么特别的理由参加游戏，只是在某个时间点开始参加了，然后一直参加下去，因为这样活着有一个方向，比没有方向好。她的师傅是一个叫白士的女性玩家，把她带入了这个世界，传授了她几乎所有的生存技巧。后来白士在第九次游戏里败给了一个叫伽罗的杀人鬼，幽鬼替她立下了一个目标：在死亡游戏中达成九十九连胜。

这个目标让她有了继续下去的理由。

她的代理人叫雪名，负责帮她处理游戏以外的所有事务——联系主办方、安排行程、管理收入，偶尔也管她有没有好好休息。雪名的户籍在参与游戏代理业之前已经被抹消，现在只用这个名字生活。

这个故事发生在幽鬼的某次游戏空窗期里。

她因为左眼旧伤复发和身体内伤，被雪名强制安置在城郊的一座欧式庄园里休养。距离下一场游戏还有三周。

然后雪名把自己的表弟也送过来了，说是来补课。

幽鬼没有明确答应过这件事。
